Getting Started

Installation

Include the plugin after jQuery and Bootstrap 5:

<!-- jQuery -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

<!-- Bootstrap 5 -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

<!-- AI Description Generator Plugin -->
<script src="path/to/ai-description-generator.js"></script>

Basic Usage

Simple Implementation

Add the data-ai-generate attribute to any textarea to enable AI generation:

<textarea data-ai-generate class="form-control" rows="5"></textarea>

Or initialize via JavaScript:

// Initialize on specific textarea
$('#my-textarea').aiDescriptionGenerator();

// Initialize on all textareas with data-ai-generate attribute
$(document).ready(function() {
    $('textarea[data-ai-generate]').aiDescriptionGenerator();
});

Advanced Usage

Configuration Options

Customize the plugin with various options:

AIGenerator.init({
    apiEndpoint: '/custom-ai-endpoint',
    i18n: {
        title: 'My AI Assistant',
        generate: 'Create Content'
    },
    styles: {
        modal: {
            maxWidth: '800px'
        }
    },
    prefill: {
        enabled: true,
        maxLength: 300
    }
});
Voice Input

Enable voice input by adding the data-ai-voice attribute:

<textarea data-ai-generate data-ai-voice="true" class="form-control" rows="5"></textarea>
HTML Output

Configure the textarea to return HTML content:

<textarea data-ai-generate data-ai-response="html" class="form-control" rows="5"></textarea>

Configuration Options

Option Type Default Description
apiEndpoint String '/api/ai/generate' Endpoint for AI generation API
apiMethod String 'POST' HTTP method for API requests
defaultMaxTokens Number 200 Default maximum length for generated content
defaultTemperature Number 0.7 Default creativity level (0-1)
prefill.enabled Boolean true Whether to prefill prompt with selected text
prefill.maxLength Number 500 Maximum characters to include from selection
prefill.append Boolean false Whether to append to existing prompt
